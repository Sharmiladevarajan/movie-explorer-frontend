# Movies Explorer Frontend

Next.js frontend with Tailwind CSS for movie management.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file from `.env.local.example`:
```bash
cp .env.local.example .env.local
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Features

- View all movies in a responsive grid
- Search movies by title, director, or description
- Add new movies
- Edit existing movies
- Delete movies
- Clean and optimized Tailwind CSS styling

## Build for Production

```bash
npm run build
npm start
```
